Claude Code: Easy Mode

One command. Beginner-friendly in seconds.

1 — Sign up for Claude (free week of Claude Code included)
2 — Open your terminal and paste this
$ curl -fsSL https://raw.githubusercontent.com/maxtattonbrown/claude-code-easy-mode/main/install.sh | bash
Installs Claude Code if you don't have it, then sets up Easy Mode.
3 — Type claude, then type /welcome
Claude will introduce itself and suggest three things to try.
🎨 Warm theme — parchment background, soft text, no black void
